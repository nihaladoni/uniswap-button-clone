## ⚡ Getting started

1. Clone the repo

```sh
git clone https://github.com/nihaladoni/uniswap-button-clone.git
```

2. Change directory

```sh
 cd uniswap-button-clone
```

3. Install NPM packages

```sh
npm install
```

4. Run the app using command

```sh
npm start
```
